# Factory Flow DBMS

Industrial Database Management & Analytics Dashboard

## 🚀 Overview

Factory Flow DBMS is a full-stack industrial management system designed to monitor production, track employee performance, analyze department efficiency, and generate intelligent operational insights.

---

## 🛠 Tech Stack

### Frontend
- React (Vite)
- Tailwind CSS
- Material UI Icons
- Framer Motion

### Backend
- Node.js
- Express
- Prisma ORM
- PostgreSQL

---

## 📊 Features

- Employee Management (CRUD + CSV Upload)
- Production Monitoring
- Department Efficiency Tracking
- Profit & Cost Analysis
- AI Analyzer
- Export Reports
- Responsive SaaS UI
